# Key bindings

## Tmux

|Syntax   |Description            |
|-------- |---------------------- |
|C-a      | prefix                |
|prefix r | reload  .tmux.conf    |
|C-g      | horizontal splits     |
|C-v      | vertical splits       |
|C-n      | create new session    |
|C-w      | create new window     |
