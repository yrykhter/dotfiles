# Key bindings for dotfiles

## Tmux

|Syntax|Description|
|-------- |---------|
|C-a|prefix|
|prefix r| reload  .tmux.conf|
|C-g| horizontal splits|
|C-v| vertical splits|
|C-n| create new session|
|C-w| create new window|

## Vim

|Syntax|Description|
|------|-------|
|C-x| open NERDTree|
|C-z| open Tagbar|

## Zsh

|Syntax|Description|
|------|-------|
|fp| open fzf search with preview|
